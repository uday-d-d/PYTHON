🐍 PYTHON Practical Programs

Welcome to the PYTHON repository!
This repository contains a collection of Python programs that I worked on during my college practical sessions.

This repo is created to:

Organize all Python practical programs in one place
Help students quickly access commonly used programs

🚀 How to Use

Clone the repository:
git clone https://github.com/your-username/PYTHON.git

Navigate to the folder:
cd PYTHON

Run any program:
python filename.py
